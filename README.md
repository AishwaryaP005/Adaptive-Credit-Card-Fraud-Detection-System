# Adaptive Credit Card Fraud Detection and Prevention System

## Overview
This project presents a hybrid fraud detection framework that combines Machine Learning, Ensemble Learning, and Reinforcement Learning techniques to detect fraudulent credit card transactions and support intelligent decision-making.

## Features
- Fraudulent transaction prediction
- Ensemble-based classification
- Reinforcement Learning for risk assessment
- Automated transaction handling
- High-accuracy fraud detection

## Dataset Features
- Transaction Amount
- Merchant Category
- Customer Location
- Transaction Time
- Additional transaction attributes

## Models Implemented

### Machine Learning Models
- Logistic Regression
- Support Vector Machine (SVM)

### Ensemble Models
- Random Forest
- Gradient Boosting
- Voting Ensemble

### Reinforcement Learning
- Q-Learning

## Results

| Model | Performance |
|---------|------------|
| Voting Ensemble | 99.6% Accuracy |
| Gradient Boosting | High Performance |
| Random Forest | High Performance |
| Logistic Regression | Baseline |
| SVM | Baseline |

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Reinforcement Learning
- Matplotlib

## Conclusion
The integration of ensemble learning and reinforcement learning significantly improved fraud detection accuracy while enabling intelligent transaction decision-making.
